## Data Visualization Project
In this data visualization project, we explored disparities in education among the different demographics in Israel in 2021. We looked at data from high school-level and university-level education, then compared them side by side.
You can explore them in this order:
- High school-level education analysis: https://public.tableau.com/app/profile/lanna.labai/viz/Project_16907479067860/HighSchoolStory?publish=yes
- University-level education analysis: https://public.tableau.com/app/profile/lanna.labai/viz/Project_16907479067860/UniversityStory?publish=yes
